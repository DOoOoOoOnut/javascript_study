## 변수 작성하기
* 변수의 이름은 문자(a~z,A~Z), 숫자(0~9), 달러($), 밑줄(_)이 포함된다.
* 변수의 이름은 시작할 수 없다.
* 변수의 이름은 대시(-)나 마침표(.)를 사용할 수 없다.
* 키워드나 예약어는 사용할 수 없다. -> function, var, class 등을 변수 이름으로 사용 불가.
* 모든 변수는 대/소문자를 구별한다, score와 Score는 다른 변수!
* 변수가 저장할 정보의 종류를 표현하는 단어를 사용해야 한다.
